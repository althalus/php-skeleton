PHP Skeleton Project

Everybody has their favourite components to use when they develop code. This is a collection of the bits and pieces I've come to desire in any PHP project I work on, and a bit of glue to help tie them together.

Composer lets me pull all of the projects in this skeleton together with ease. Make sure composer is installed, and just run composer.phar install

Idiorm and Paris provide a nice abstraction for SQL databases. Small, simple, stays out of the way and lets me do my job. We use a forked git repo that has composer.json files for easier composer magic.

Twig provides a template framework. Nicer, safer and easier than trying to do <?php echo $var; ?> everywhere.

Slim for a nice routing framework

Phorms for some nice forms

And lastly, the Twitter Bootstrap toolkit.
